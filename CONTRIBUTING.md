# Contributing

1.  Fork the repo.
2.  Create a feature branch: `git checkout -b feature/my-cool-feature`.
3.  Commit changes: `git commit -m "feat: add cool feature"`.
4.  Push to branch: `git push origin feature/my-cool-feature`.
5.  Open a Pull Request.

## Standards
- **Commits**: Conventional Commits (feat, fix, docs, chore).
- **Linting**: Ensure `pnpm lint` passes.
- **Tests**: Ensure `make test` passes.
